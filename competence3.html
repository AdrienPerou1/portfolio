Installation de services réseau (cliquer pour dérouler)

Cette SAÉ vise à installer et configurer un serveur web avec une authentification sécurisé.

Langages et outils utilisés

  • Machine virtuelle sur Linux : Pour l'environnement serveur.
  • Apache : Serveur web.
  • MySQL : Système de gestion de base de données.
  • PHP : Langage de script côté serveur.

Description du projet

Le projet consistait à installer et configurer un serveur web complet avec Apache, MySQL et PHP. Voici quelques-unes des tâches réalisées :

  • Installation et configuration du serveur Apache
  • Configuration de la base de données MySQL
  • Intégration de PHP pour le traitement dynamique des pages web

Exemple de configuration Apache


# Chargement des modules nécessaires
LoadModule php7_module modules/libphp7.so

# Configuration de la racine des documents
DocumentRoot "/var/www/html"

    Options Indexes FollowSymLinks
    AllowOverride All
    Require all granted

Automatisation de scripts PHP avec Docker (cliquer pour dérouler)

Cette SAÉ se concentre sur l'automatisation de scripts PHP avec Docker. Le projet impliquait la création d'un script PHP capable de transformer des fichiers Markdown en PDF.

Langages et outils utilisés

  • PHP : Pour le développement du script.
  • Docker : Pour la conteneurisation du script.

Description du projet

Le projet comprenait plusieurs étapes :

  • Développement d'un script PHP pour transformer des fichiers Markdown en PDF
  • Automatisation du processus de transformation avec Docker

Exemple de script PHP


<?php
require 'vendor/autoload.php';

use Spatie\PdfToText\Pdf;
use League\CommonMark\CommonMarkConverter;

$markdown = file_get_contents('example.md');

$converter = new CommonMarkConverter();
$html = $converter->convertToHtml($markdown);

$pdf = new Pdf('/usr/local/bin/pdftotext');
$pdf->setPdf($html);
$pdf->saveAs('output.pdf');
?>
