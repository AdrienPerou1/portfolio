Projet : Création graphique et développement du site web "Tabas'KHO"

Création d'une charte graphique avec Canva (cliquer pour dérouler)

Cette partie du projet consistait à créer une charte graphique complète pour le site web de streaming "Tabas'KHO" en utilisant Canva.

Outils utilisés

  • Canva : Pour la création de la charte graphique.

Description de la charte graphique

Les éléments inclus dans la charte graphique sont :

  • Logo de l'entreprise
  • Palette de couleurs
  • Typographie
  • Icônes et images

Création d'une maquette de site web avec Figma (cliquer pour dérouler)

Cette partie du projet consistait à créer une maquette interactive du site web de streaming "Tabas'KHO" en utilisant Figma.

Outils utilisés

  • Figma : Pour la création de la maquette interactive.

Description de la maquette

Les éléments inclus dans la maquette sont :

  • Page d'accueil
  • Page de catalogue des films
  • Page de détail d'un film
  • Page de connexion et d'inscription

Développement d'une page HTML/CSS pour le site "Tabas'KHO" (cliquer pour dérouler)

Cette partie du projet consistait à développer une page HTML/CSS pour le site web de streaming "Tabas'KHO" en utilisant les normes de la charte graphique et de la maquette.

Outils et langages utilisés

  • HTML : Pour la structure du site web.
  • CSS : Pour le style et la mise en page.

Description du développement web

Le projet incluait les éléments suivants :

  • Structure de la page d'accueil
  • Style conforme à la charte graphique
  • Utilisation des classes et identifiants CSS pour la mise en page
