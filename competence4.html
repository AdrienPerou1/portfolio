Projets liés à la gestion de données

Création d'un diagramme UML et base de données PostgreSQL (cliquer pour dérouler)

Cette SAÉ consistait à créer un diagramme de classe UML avec Visual Paradigm, puis à implémenter la base de données en PostgreSQL avec SQL Workbench.

Langages et outils utilisés

  • Visual Paradigm : Pour la création du diagramme UML.
  • PostgreSQL : Système de gestion de base de données.
  • SQL Workbench : Pour la gestion et la manipulation de la base de données.

Description du projet

Le projet incluait les étapes suivantes :

  • Création d'un diagramme de classe UML détaillant les entités et leurs relations
  • Implémentation du modèle de données en PostgreSQL

Exemple de script SQL pour la création des tables


CREATE TABLE Utilisateur (
    id SERIAL PRIMARY KEY,
    nom VARCHAR(50),
    email VARCHAR(100) UNIQUE NOT NULL
);

CREATE TABLE Produit (
    id SERIAL PRIMARY KEY,
    nom VARCHAR(100),
    prix DECIMAL
);

CREATE TABLE Commande (
    id SERIAL PRIMARY KEY,
    utilisateur_id INT REFERENCES Utilisateur(id),
    date_commande DATE
);

Création de tables SQL et analyse avec Python (cliquer pour dérouler)

Cette SAÉ impliquait la création de tables SQL à partir d'un diagramme UML, le peuplement des tables et l'analyse des statistiques avec Python.

Langages et outils utilisés

  • SQL : Pour la création et le peuplement des tables.
  • Python : Pour l'analyse des données et les statistiques.

Description du projet

Les tâches réalisées comprenaient :

  • Développement du script SQL pour créer les tables et les relations
  • Peuplement des tables avec des données pertinentes
  • Utilisation de Python pour analyser les données et générer des statistiques

Exemple de script SQL pour le peuplement des tables


INSERT INTO Utilisateur (nom, email) VALUES ('Alice', 'alice@example.com');
INSERT INTO Utilisateur (nom, email) VALUES ('Bob', 'bob@example.com');

INSERT INTO Produit (nom, prix) VALUES ('Ordinateur', 1000.00);
INSERT INTO Produit (nom, prix) VALUES ('Souris', 20.00);

INSERT INTO Commande (utilisateur_id, date_commande) VALUES (1, '2024-01-15');
INSERT INTO Commande (utilisateur_id, date_commande) VALUES (2, '2024-01-20');

Exemple de script Python pour l'analyse


import psycopg2
import pandas as pd

# Connexion à la base de données PostgreSQL
conn = psycopg2.connect(
    dbname="votre_base_de_donnees",
    user="votre_utilisateur",
    password="votre_mot_de_passe",
    host="localhost",
    port="5432"
)

# Requête SQL pour récupérer les données
query = "SELECT * FROM Commande JOIN Utilisateur ON Commande.utilisateur_id = Utilisateur.id"
data = pd.read_sql_query(query, conn)

# Analyse des données avec pandas
statistiques = data.groupby('nom')['date_commande'].count()

print(statistiques)
